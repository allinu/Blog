#运维 #安全

## 放行端口

1. 查看当前命令

```bash
sudo iptables -L
```

2. 放行`80`端口

```bash
sudo iptables -I INPUT -p tcp --dport 80 -m state --state NEW -j ACCEPT
```

3. 保存规则

```bash
sudo iptables-legacy-save
```
